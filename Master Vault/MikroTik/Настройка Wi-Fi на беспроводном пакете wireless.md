В данном примере рассматривается точка доступа MikroTik с пакетом `wireless`, поддерживающим Wi-Fi 5 (`802.11a/b/g/n/ac`) и два диапазона: `2,4 ГГц` и `5 ГГц`.

Некоторые параметры на разных беспроводных пакетах могут иметь разные значения или на одном пакете могут быть использованы, а на другом нет. Это связано со спецификой данных параметров на разных пакетах. Например:

- На пакете `wifi-qcom` параметр `Mode` имеет значение ap, а на пакете `wireless` – `ap bridge`. Эти разные значения являются идентичными, но имеют разные названия на разных беспроводных пакетах;
- На пакете `wifi-qcom` параметр `Installation` не задан, а на пакете wireless имеет значение `indoor`. Это связано с тем, что на пакете `wifi-qcom` данный параметр по умолчанию имеет значение `indoor` и его не надо задавать явным образом, а на пакете `wireless` этот же параметр по умолчанию имеет значение any и поэтому для этого параметра явным образом указано значение `indoor`.

### Настройка интерфейса 2,4 ГГц

Диапазон `2,4 ГГц` рекомендуется использовать только при необходимости, так как он сильно загружен.

Используемые параметры:

- имя интерфейса: `wlan‑2GHz` (интерфейс переименован для удобства его идентификации при анализе конфигурации);
- имя беспроводной сети (SSID): `AP‑2`;
- способ аутентификации: `WPA2-PSK`;
- пароль: `password`;
- функция WPS: `отключена` (включение повышает уязвимость сети Wi-Fi);
- поддержка WMM: `включена`;
- поддерживаемые поправки: `IEEE 802.11b/g/n`;
- использование только стандартных центральных частот: `включено`;
- ширина канала: `20 МГц`;
- использование кадров RTS и CTS: `включено`;
- региональные ограничения: `superchannel` и `no_country-set`;
- допустимый диапазон центральных частот: `2412–2472 МГц`;
- максимальная мощность передатчика: `18 дБм`.

### Настройка интерфейса 5 ГГц

Диапазон `5 ГГц` рекомендуется использовать как основной.

Используемые параметры:

- имя интерфейса: `wlan‑5GHz` (интерфейс переименован для удобства его идентификации при анализе конфигурации);
- имя беспроводной сети (SSID): `AP‑5`;
- способ аутентификации: `WPA2-PSK`;
- пароль: `password`;
- функция WPS: `отключена` (включение повышает уязвимость сети Wi-Fi);
- поддержка WMM: `включена`;
- поддерживаемые поправки: `IEEE 802.11a//n/ac`;
- использование только стандартных центральных частот: `включено`;
- ширина канала: `40 МГц`;
- использование кадров RTS и CTS: `включено`;
- региональные ограничения: `superchannel` и `no_country-set`;
- допустимые диапазоны центральных частот: `5180–5240, 5260–5320, 5660–5720, 5745–5825 МГц`;
- максимальная мощность передатчика: `21 дБм`.

### Примечания

- Причины, по которым сети Wi-Fi для диапазонов `2,4 ГГц` и `5 ГГц` имеют разные имена и пароли, описаны в разделе «Настройки, положительно влияющие на производительность беспроводной сети».
- Использование ширины канала `20 МГц` для `2,4 ГГц` и `40 МГц` для `5 ГГц` соответствует общепринятой практике и помогает снизить помехи для соседних сетей Wi-Fi.
- Режимы `Superchannel` и `no_country_set` фактически отключают региональные ограничения по частотам и мощности передатчика. Это используется из-за особенностей RouterOS и некоторых устройств MikroTik. Если `Superchannel` отсутствует или при его выборе появляется ошибка `failed to set country`, рекомендуется переустановить RouterOS через Netinstall.
- При использовании устройства вне РФ необходимо выбирать частоты в соответствии с местным законодательством.
- Если коэффициент усиления антенны для `2,4 ГГц` отличается от `1,5 dBi`, мощность передатчика необходимо пересчитать.
- Для `2,4 ГГц` в настройках указывается `15 dBm`, а не `18 dBm`. Это связано с особенностями пакета `wireless`.
- Если коэффициент усиления антенны для `5 ГГц` отличается от `2 dBi`, мощность передатчика необходимо пересчитать.
- Для `5 ГГц` в настройках указывается `21 dBm`, а не `18 dBm`. Это связано с особенностями пакета `wireless`.

### Расчет мощности для 2,4 ГГц

Максимальная мощность рассчитывается по формуле:

EIRP−G=Ptx

В РФ максимальная ЭИИМ для `2,4 ГГц` составляет `20 dBm`.

При усилении антенны `1,5 dBi` значение округляется до `2 dBi`:

20 dBm − 2 dBi = 18 dBm

В пакете `wireless` для `802.11n` мощность указывается отдельно для каждого радиоканала. Так как используется два канала, итоговое значение уменьшается на `3 dB`:

18 dBm − 3 dB = 15 dBm

```fold title:"Configure 2.4 GHz Wi-Fi"
/interface/wireless/security-profiles/
set [ find default=yes ] supplicant-identity=MikroTik
add authentication-types=wpa2-psk mode=dynamic-keys name=profile_2ghz \
   supplicant-identity="" wpa2-pre-shared-key=password1

/interface/wireless/
set [ find default-name=wlan1 ] band=2ghz-onlyn country=no_country_set \
   disabled=no frequency=auto frequency-mode=superchannel hw-protection-mode=rts-cts \
   installation=indoor mode=ap-bridge radio-name=AP name=wlan-2GHz scan-list=2412-2472 \
   security-profile=profile_2ghz ssid=AP-2 tx-power=15 tx-power-mode=all-rates-fixed \
   wireless-protocol=802.11 wmm-support=enabled wps-mode=disabled
```

## Настройка Wi-Fi 5 ГГц

Для диапазона `5 ГГц` используются следующие параметры:

- интерфейс: `wlan-5GHz`;
- SSID: `AP-5`;
- защита: `WPA2-PSK`;
- WPS отключен;
- WMM включен;
- ширина канала: `20/40 МГц`;
- диапазоны частот: `5180–5240`, `5260–5320`, `5660–5720`, `5745–5825 МГц`;
- RTS/CTS включен;
- мощность: `21 dBm`.

### Расчет мощности для 5 ГГц

В РФ максимальная ЭИИМ для `5 ГГц` составляет `23 dBm`.

При усилении антенны `2 dBi` расчет будет следующим:

23 dBm − 2 dBi = 21 dBm

Для `802.11ac` в пакете `wireless` параметр `tx-power` задает суммарную мощность, поэтому дополнительное уменьшение мощности не требуется.

```fold title:"Configure 5 GHz Wi-Fi"
/interface/wireless/security-profiles/
set [ find default=yes ] supplicant-identity=MikroTik
add authentication-types=wpa2-psk mode=dynamic-keys name=profile_5ghz \
   supplicant-identity="" wpa2-pre-shared-key=password2

/interface/wireless/
set [ find default-name=wlan2 ] band=5ghz-onlyac channel-width=20/40mhz-XX \
   country=no_country_set disabled=no frequency=auto frequency-mode=superchannel \
   hw-protection-mode=rts-cts installation=indoor mode=ap-bridge radio-name=AP name=wlan-5GHz \
   scan-list=5180-5240,5260-5320,5660-5720,5745-5825 security-profile=profile_5ghz \
   ssid=AP-5 tx-power=21 tx-power-mode=all-rates-fixed wireless-protocol=802.11 \
   wmm-support=enabled wps-mode=disabled
```

В данном примере используется точка доступа MikroTik с пакетом `wifi-qcom`, поддерживающим Wi-Fi 6 (`802.11ax`) и два диапазона: `2,4 ГГц` и `5 ГГц`.

Настройка пакетов `wifi-qcom` и `wifi-qcom-ac` практически одинаковая. Разница в том, что `wifi-qcom-ac` не поддерживает функции Wi-Fi 6.

Некоторые параметры на разных беспроводных пакетах могут иметь разные значения или на одном пакете могут быть использованы, а на другом нет. Это связано со спецификой данных параметров на разных пакетах. Например:

- На пакете `wifi-qcom` параметр `Mode` имеет значение ap, а на пакете `wireless` – `ap bridge`. Эти разные значения являются идентичными, но имеют разные названия на разных беспроводных пакетах;
- На пакете `wifi-qcom` параметр `Installation` не задан, а на пакете `wireless` имеет значение `indoor`. Это связано с тем, что на пакете `wifi-qcom` данный параметр по умолчанию имеет значение `indoor` и его не надо задавать явным образом, а на пакете `wireless` этот же параметр по умолчанию имеет значение `any` и поэтому для этого параметра явным образом указано значение `indoor`.;
- На пакете `wifi-qcom-ac` параметр `Country` не может иметь значение `Superchannel`, а на `wifi-qcom` – может.

### Настройка интерфейса 2,4 ГГц

Диапазон `2,4 ГГц` рекомендуется использовать только при необходимости, так как он сильно загружен.

Используемые параметры:

- имя интерфейса: `wlan‑2GHz` (интерфейс переименован для удобства его идентификации при анализе конфигурации);
- имя беспроводной сети (SSID): `AP‑2`;
- способы аутентификации: `WPA2-PSK` и `WPA3-PSK`;
- пароль: `password`;
- функция WPS: `отключена` (включение повышает уязвимость сети Wi-Fi);
- поддерживаемые поправки: `IEEE 802.11b/g/n/ax`
- ширина канала: `20 МГц`;
- использование кадров RTS и CTS: `включено` (только для пакета wifi-qcom);
- время повторного поиска наименее загруженной частоты: `от 2 до 4 часов`;
- региональные ограничения для пакета wifi-qcom: `Superchannel`;
- региональные ограничения для пакета wifi-qcomp-ac: `Russia`;
- допустимый диапазон центральных частот: `2412–2472 МГц`;
- максимальная мощность передатчика: `16 дБм`.

### Настройка интерфейса 5 ГГц

Диапазон `5 ГГц` рекомендуется использовать как основной.

Используемые параметры:

- имя интерфейса: `wlan‑5GHz` (интерфейс переименован для удобства его идентификации при анализе конфигурации);
- имя беспроводной сети (SSID): `AP‑5`;
- способы аутентификации: `WPA2-PSK` и `WPA3-PSK`;
- пароль: `password`;
- функция WPS: `отключена` (включение повышает уязвимость сети Wi-Fi);
- поддерживаемые поправки: `IEEE 802.11a//n/ac/ax`;
- ширина канала: `40 МГц`;
- использование кадров RTS и CTS: `включено` (только для пакета wifi-qcom);
- время повторного поиска наименее загруженной частоты: от 2 до 4 часов;
- региональные ограничения для пакета wifi-qcom: `Superchannel`;
- региональные ограничения для пакета wifi-qcomp-ac: `Russia`;
- допустимые диапазоны центральных частот: `5180–5240, 5260–5320, 5660–5720, 5745–5825 МГц`;
- максимальная мощность передатчика: `18 дБм`.

### Примечания

- Причины, по которым сети Wi-Fi для диапазонов `2,4 ГГц` и `5 ГГц` имеют разные имена и пароли, описаны в разделе «Настройки, положительно влияющие на производительность беспроводной сети».
- Использование ширины канала `20 МГц` для `2,4 ГГц` и `40 МГц` для `5 ГГц` соответствует общепринятой практике и помогает снизить помехи для соседних сетей Wi-Fi.
- Режим `Superchannel` фактически отключает региональные ограничения по частотам и мощности передатчика. Это используется из-за особенностей RouterOS и некоторых устройств MikroTik.
- Если на устройстве с пакетом `wifi-qcom` отсутствует режим `Superchannel` или при его выборе появляется ошибка `failed to set country`, рекомендуется переустановить RouterOS через Netinstall.
- На устройствах с пакетом `wifi-qcom-ac` режим `Superchannel` не поддерживается. В этом случае для РФ следует использовать регион `Russia`.
- При использовании устройства вне РФ необходимо выбирать частоты в соответствии с местным законодательством.
- Если коэффициент усиления антенны для `2,4 ГГц` отличается от `4 dBi`, мощность передатчика необходимо пересчитать.
- Если коэффициент усиления антенны для `5 ГГц` отличается от `4,5 dBi`, мощность передатчика необходимо пересчитать.

### Расчет мощности передатчика для 2,4 ГГц

Максимальная мощность передатчика (`Max Tx Power`) зависит от законодательных ограничений и коэффициента усиления антенны.

Расчет выполняется по формуле:

EIRP−G=Ptx​

где:
- `EIRP` — максимально допустимая ЭИИМ;
- `G` — коэффициент усиления антенны;
- `Ptx` — мощность передатчика.

В РФ для диапазона `2,4 ГГц` максимальная ЭИИМ составляет `20 dBm`.

Если коэффициент усиления антенны равен `4 dBi`, то расчет будет следующим:

20 dBm − 4 dBi = 16 dBm

Таким образом, значение `Max Tx Power` должно быть установлено в `16 dBm`.

```fold title:"Configure 2.4 GHz Wi-Fi"
/interface/wifi/
set [ find default-name=wifi2 ] channel.band=2ghz-ax .frequency=2412-2472 \
   .reselect-interval=2h..4h .width=20mhz configuration.country=Superchannel \
   .hw-protection-mode=rts-cts .mode=ap .ssid=AP-2 .tx-power=16 disabled=no name=wlan-2GHz \
   security.authentication-types=wpa2-psk,wpa3-psk .passphrase=password1 .wps=disable
```

### Расчет мощности передатчика для 5 ГГц

Максимальная мощность передатчика для диапазона `5 ГГц` рассчитывается по формуле:

EIRP−G=Ptx​

В РФ максимальная ЭИИМ для диапазона `5 ГГц` составляет `23 dBm`.

Если коэффициент усиления антенны равен `4,5 dBi`, значение округляется до `5 dBi`. Тогда расчет будет следующим:

23 dBm − 5 dBi = 18 dBm

Таким образом, значение `Max Tx Power` должно быть установлено в `18 dBm`.

Примечания:
- в пакете `wifi-qcom-ac` нельзя использовать `Country=Superchannel`;
- параметр `Hw. Protection Mode` поддерживается только пакетом `wifi-qcom`.

```fold title:"Configure 5 GHz Wi-Fi"
/interface/wifi/
set [ find default-name=wifi2 ] channel.band=2ghz-ax .frequency=2412-2472 \
   .reselect-interval=2h..4h .width=20mhz configuration.country=Superchannel \
   .hw-protection-mode=rts-cts .mode=ap .ssid=AP-2 .tx-power=16 disabled=no name=wlan-2GHz \
   security.authentication-types=wpa2-psk,wpa3-psk .passphrase=password1 .wps=disable
```

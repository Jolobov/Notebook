## Виды беспроводных пакетов на устройствах MikroTik

Пакеты (`.npk`) в RouterOS добавляют отдельные функции системы. Основной пакет обязателен для работы устройства, дополнительные устанавливаются по необходимости.

За работу Wi-Fi отвечают отдельные беспроводные пакеты:
- `wireless` — устаревший пакет для старых устройств;
- `wifi-qcom-ac` — пакет для устройств с Wi-Fi 5 (`802.11ac`);
- `wifi-qcom` — пакет для современных устройств с Wi-Fi 6 (`802.11ax`).

При обновлении RouterOS автоматически обновляются все установленные пакеты, включая Wi-Fi-пакеты.

## Настройка Wi-Fi на беспроводных пакетах wifi-qcom и wifi-qcom-ac

В данном примере используется точка доступа MikroTik с пакетом `wifi-qcom`, поддерживающим Wi-Fi 6 (`802.11ax`) и два диапазона: `2,4 ГГц` и `5 ГГц`.

Настройка пакетов `wifi-qcom` и `wifi-qcom-ac` практически одинаковая. Разница в том, что `wifi-qcom-ac` не поддерживает функции Wi-Fi 6.

Некоторые параметры могут отличаться в зависимости от используемого пакета. Например:
- в `wifi-qcom` режим точки доступа называется `ap`, а в `wireless` — `ap bridge`;
- параметр `Installation` в `wifi-qcom` обычно не требуется задавать вручную;
- `Superchannel` доступен только в `wifi-qcom`.

### Настройка Wi-Fi 2,4 ГГц

Диапазон `2,4 ГГц` рекомендуется использовать только при необходимости, так как он сильно загружен.

Используемые параметры:
- интерфейс: `wlan-2GHz`;
- SSID: `AP-21`;
- защита: `WPA2-PSK` и `WPA3-PSK`;
- ширина канала: `20 МГц`;
- WPS отключен;
- автоматический выбор частоты: `2–4 часа`;
- диапазон частот: `2412–2472 МГц`;
- мощность: `16 dBm`.

### Настройка Wi-Fi 5 ГГц

Диапазон `5 ГГц` рекомендуется использовать как основной.

Используемые параметры:
- интерфейс: `wlan-5GHz`;
- SSID: `AP-51`;
- защита: `WPA2-PSK` и `WPA3-PSK`;
- ширина канала: `40 МГц`;
- WPS отключен;
- автоматический выбор частоты: `2–4 часа`;
- диапазоны частот: `5180–5240`, `5260–5320`, `5660–5720`, `5745–5825 МГц`;
- мощность: `18 dBm`.

### Примечания

- Для диапазонов `2,4 ГГц` и `5 ГГц` рекомендуется использовать разные SSID.
- Более широкие каналы не всегда дают прирост скорости, но могут создавать помехи.
- Режим `Superchannel` фактически отключает региональные ограничения и используется из-за особенностей RouterOS и драйверов MikroTik.
- При использовании устройства вне РФ необходимо выбирать частоты и мощность в соответствии с местным законодательством.

### Расчет мощности передатчика для 2,4 ГГц

Максимальная мощность передатчика (`Max Tx Power`) зависит от законодательных ограничений и коэффициента усиления антенны.

Расчет выполняется по формуле:

EIRP−G=Ptx​

где:
- `EIRP` — максимально допустимая ЭИИМ;
- `G` — коэффициент усиления антенны;
- `Ptx` — мощность передатчика.

В РФ для диапазона `2,4 ГГц` максимальная ЭИИМ составляет `20 dBm`.

Если коэффициент усиления антенны равен `4 dBi`, то расчет будет следующим:

20 dBm − 4 dBi = 16 dBm

Таким образом, значение `Max Tx Power` должно быть установлено в `16 dBm`.






```fold title:""
/interface/wifi/
set [ find default-name=wifi2 ] channel.band=2ghz-ax .frequency=2412-2472 \
   .reselect-interval=2h..4h .width=20mhz configuration.country=Superchannel \
   .hw-protection-mode=rts-cts .mode=ap .ssid=AP-2 .tx-power=16 disabled=no name=wlan-2GHz \
   security.authentication-types=wpa2-psk,wpa3-psk .passphrase=password1 .wps=disable
```
